# Silmeria Project

### 简介

这是前端项目，使用 AngularJS 作为前端框架。

### 使用方法

```bash
# install dependency
$ npm install -g gulp bower
$ cd Silmeria
$ npm install
$ bower install

# run server
$ gulp server

# build the project
$ gulp build
```
