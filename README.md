# Brynhild Project

### 简介

这是前端项目，使用 AngularJS 作为前端框架，利用 api 进行通讯。

### 使用方法

```bash
$ npm install -g grunt-cli bower
$ cd Brynhild
$ npm install
$ bower install

# 运行开发环境
$ grunt server
```
